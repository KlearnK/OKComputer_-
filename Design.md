## 全局动效系统

### 动画时序

**缓动函数库:**
```css
:root {
  --ease-out-expo: cubic-bezier(0.16, 1, 0.3, 1);
  --ease-out-quart: cubic-bezier(0.165, 0.840, 0.440, 1);
  --ease-elastic: cubic-bezier(0.68, -0.55, 0.265, 1.55);
  --ease-smooth: cubic-bezier(0.4, 0, 0.2, 1);
  --ease-dramatic: cubic-bezier(0.87, 0, 0.13, 1);
  --ease-bounce: cubic-bezier(0.34, 1.56, 0.64, 1);
}
```

**时长比例:**
- 微型: 150ms (悬停、切换)
- 快速: 300ms (按钮、链接)
- 正常: 500ms (卡片、显示)
- 慢速: 800ms (区块入场)
- 戏剧性: 1200ms (主视觉、重要揭示)

**交错模式:**
- 级联: 兄弟元素间隔80ms
- 波浪: 120ms带正弦偏移
- 爆发: 50ms快速连发

### 持续效果

**区块特定环境动效:**
- 主视觉区: 浮动目标粒子（微妙，最多15个）
- 统计区: 数字微光效果
- 目标区块: 进度条呼吸动画
- 页脚: 社交图标轨道环

**有目的的动态:**
- 无随机几何装饰
- 所有动效服务于导航或强调内容
- 粒子系统仅用于庆祝时刻

### 滚动引擎配置

**视差层:**
```javascript
const parallaxConfig = {
  hero: { speed: 0.3, direction: 'vertical' },
  cards: { speed: 0.1, direction: 'vertical' },
  decorative: { speed: 0.5, direction: 'both' }
};
```

**固定点:**
- 主视觉区: 固定额外50vh滚动
- 统计区: 固定用于计数动画
- 目标区: 固定用于水平滚动展示

## 技术实现说明

### 所需库
- **GSAP + ScrollTrigger**: 复杂滚动动画和固定
- **Intersection Observer**: 触发动画的原生API
- **CSS自定义属性**: 动态主题和动画值
- **Three.js**（可选）: 仅当实现3D图表时需要

### 性能优化

**关键规则:**
- 所有动画使用`transform`和`opacity`
- 仅在动画前立即应用`will-change`，动画后移除
- 对动画区块使用CSS containment
- 滚动事件节流至16ms（60fps）
- 使用Intersection Observer进行可见性触发
- 懒加载图片和重型组件

**GPU加速:**
```css
.animated-element {
  transform: translateZ(0);
  backface-visibility: hidden;
}
```

**鼠标效果:**
- 所有鼠标追踪使用CSS `:hover`变换
- 绝不在mousemove事件中使用setState
- 使用CSS自定义属性通过节流JS更新（最大60fps）

### 浏览器支持
- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 对旧浏览器优雅降级
- 使用CSS.supports()进行特性检测
- 必要时提供polyfill

## 动画数值总结

### 运动幅度
- 入场滑动: 30-100px
- 缩放效果: 0.8x - 1.2x
- 旋转: 5° - 15°（特殊时刻最大360°）
- 视差范围: 40px - 100px
- 3D深度（translateZ）: 20px - 200px

### 时序总结
- 微型: 150ms
- 快速: 300ms
- 正常: 500ms
- 慢速: 800ms
- 戏剧性: 1200ms

### 缓动总结
- 入场: `cubic-bezier(0.16, 1, 0.3, 1)` (ease-out-expo)
- 悬停: `cubic-bezier(0.4, 0, 0.2, 1)` (ease-smooth)
- 弹性: `cubic-bezier(0.68, -0.55, 0.265, 1.55)` (ease-elastic)
- 平滑: `cubic-bezier(0.165, 0.840, 0.440, 1)` (ease-out-quart)
